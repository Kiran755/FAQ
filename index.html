Is competetive coding necessary for getting placements?

It depends. You may not need CP if you already are great in other things like Fronted, fullstack, android,blockchain or machine learning etc. Competetive coding is not easy, because its not just competition, Its a sport. People from accross the world participate in it on a daily basis.If you really enjoy coding then you might as well do CP for fun. But if you are not good at coding and want to try and explore different things then you should try other options like webdev, android development,etc. Hence, it depends on the person but it is not necessary to do CP for getting placements.
